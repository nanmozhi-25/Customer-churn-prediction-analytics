# вљ™пёЏ Installation

## 1. Clone Repository

```bash
git clone https://github.com/nanmozhi-25/Customer-churn-prediction-analytics.git
```

```bash
cd Customer-churn-prediction-analytics
```

---

## 2. Install Backend

```bash
cd server

npm install

npm start
```

Server runs on

```
http://localhost:5000
```

---

## 3. Install Frontend

Open another terminal

```bash
cd client

npm install

npm run dev
```

Frontend runs on

```
http://localhost:5173
```

---

## 4. Run ML Service (Optional)

```bash
cd ml-service

pip install -r requirements.txt

python app.py
```

ML Service runs on

```
http://localhost:8000
```
